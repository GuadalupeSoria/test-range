Range

Custom range slider component built with Next.js and TypeScript.. The same component handles two modes depending on the props it receives.

Two exercises:
- Exercise 1 — normal range with editable min/max labels, values come from an API
- Exercise 2 — fixed values only, no text input, values come from an API

 Install

pnpm install


pnpm dev

Open http://localhost:8080

Test

pnpm test
